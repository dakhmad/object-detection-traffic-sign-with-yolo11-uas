## Dataset
Dataset yang digunakan dalam tugas projek ini berasal dari Kaggle dan
dilabeli menggunakan Roboflow. Dataset tidak disertakan dalam repository
karena keterbatasan ukuran penyimpanan GitHub.

Dataset dapat diakses melalui:
- Kaggle: https://www.kaggle.com/datasets/cakrulgaming/indonesia-traffic-sign
- Roboflow: https://app.roboflow.com/ds/JpmegV7gTA?key=MDK7wbYZum

Dataset diekspor dalam format YOLO8 dan digunakan pada proses training
dan evaluasi model.

Struktur folder berada dalam format YOLO8 seperti berikut.
